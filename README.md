# 🚀 Custom System FPS Analizer v7.0

Bu araç, bilgisayar donanımınıza ve seçtiğiniz grafik segmentine (Piksel, 2D, 3D vb.) göre alacağınız tahmini FPS değerini analiz eden Python tabanlı bir yazılımdır.

## ✨ Özellikler
- **Gerçekçi Donanım Seçimi:** Intel i3/i5/i7 ve AMD R3/R5/R7 desteği.
- **Grafik Kategorileri:** Oyun isimleri yerine grafik türüne göre (3D Ağır, Orta vb.) analiz.
- **Çoklu Dil:** Türkçe ve İngilizce dil desteği.
- **Hata Koruması:** Kullanıcı hatalarına karşı dayanıklı (Foolproof) yapı.

## 🛠 Terminalden İndirme ve Çalıştırma

Projeyi bilgisayarınıza indirmek için terminale şu komutu yazın:

```bash
git clone [https://github.com/emircls3437-hub/Custom-System-FPS-Analizer.git](https://github.com/emircls3437-hub/Custom-System-FPS-Analizer.git)
cd Custom-System-FPS-Analizer
python3 analizer.py
